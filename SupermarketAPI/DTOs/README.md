# Supermarket ERP System

A full-stack supermarket management solution with role-based dashboards for Admin and Employee users.

## 🌐 Live Demo
- Login: https://web-three-alpha-80.vercel.app/#/login
- Admin Dashboard: https://web-three-alpha-80.vercel.app/#/admin
- Employee Dashboard: https://web-three-alpha-80.vercel.app/#/employee

## 🚀 Features

- 🔐 Role-based authentication (Admin & Employee)
- 📦 Inventory management
- 🛒 Billing system
- 📊 Dashboard with analytics
- 🧾 Product management
- 📈 Reporting system

---

## 🛠️ Tech Stack

- Frontend: Flutter
- Backend: .NET
- Database: MySQL
- Deployment: Vercel, Render, Azure
